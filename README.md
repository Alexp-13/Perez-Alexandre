# Portfolio

This repository hosts a personal portfolio built with React, TypeScript, and Vite. It showcases projects, skills, and contact information in a clean and interactive web experience.

## 🚀 Getting Started

Clone the repository and install dependencies:

```bash
bun install
```

Run the development server:

```bash
bun run dev
```

Build for production:

```bash
bun run build
```

Preview the production build:

```bash
bun run preview
```

## 📁 Structure

- `src/` – React components, styles, and assets used in the portfolio.
- `public/` – Static files served by Vite.
- `content/` – Any additional content or data feeds.

## 💡 Features

- Responsive design with CSS animations and interactive elements.
- Supercharged with custom components like `Navbar`, `ProjectModal`, and `NeuralNetworkCanvas`.
- Easy to update projects and information via the `src/data` directory.

Feel free to reach out via email or social links included in the `Contact` component.
